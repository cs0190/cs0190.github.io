3.4 Booleans

Boolean
The type of the values true and false.

3.4.1 Boolean Functions

not :: (b :: Boolean) -> Boolean

Returns true when given false and vice versa.

Examples:
check:
  not(true) is false
  not(2 < 1) is true
end
